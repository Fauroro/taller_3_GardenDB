1. **Primera Forma Normal (1NF)**:
   - Se crearon tablas separadas para entidades distintas, como producto, proveedor, cliente, empleado, etc. Esto garantiza que cada tabla tenga un formato simple y que contenga solo atributos relevantes para esa entidad.
   - Se asegur√≥ que cada celda de cada tabla contenga un √∫nico valor, evitando la redundancia y la repetici√≥n de datos.
2. **Segunda Forma Normal (2NF)**:
   - Se identificaron y eliminaron las dependencias parciales en las tablas. Por ejemplo:
     - Se cre√≥ la tabla gama_producto para almacenar informaci√≥n sobre las gamas de productos, eliminando la dependencia parcial de la tabla producto con respecto a la descripci√≥n de la gama.
     - Se cre√≥ la tabla dimensiones para almacenar informaci√≥n sobre las dimensiones de los productos, eliminando la dependencia parcial de la tabla producto con respecto a las dimensiones.
3. **Tercera Forma Normal (3NF)**:
   - Se identificaron y eliminaron las dependencias transitivas en las tablas. Por ejemplo:
     - Se separaron los detalles espec√≠ficos del proveedor en la tabla proveedor, eliminando la dependencia de la tabla producto con respecto a estos detalles.
     - Se cre√≥ la tabla puesto_empleado para almacenar informaci√≥n sobre los puestos de los empleados, eliminando la dependencia de la tabla empleado con respecto a esta informaci√≥n.
